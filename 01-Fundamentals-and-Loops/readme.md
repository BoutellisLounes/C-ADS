# Sub-Project 01: My First Steps in C & Control Loops

## Overview
In this first module, I started my journey into programming by setting up my coding environment and learning the basic rules of logic. I focused on understanding variables, conditions, and different types of loops to solve practical problems.

## What I Did
* **Environment Setup:** I learned how to use the Dev-C++ IDE to write, compile, and debug my very first C programs, starting with a classic "Hello World" script.
* **Basic Calculations:** I wrote simple programs to swap variable values, reverse three-digit numbers, and calculate the area of a circle.
* **Conditions & Logic:** I used `if-else` blocks to find the minimum of two numbers, check if an integer is even or odd, and built a simple 4-operation calculator. I also wrote a program to solve quadratic equations using the `math.h` library.
* **Loops & Repetition:** I practiced using `for`, `while`, and `do-while` loops by calculating factorials, checking for perfect numbers, and printing multiplication tables based on user input.
* **Pattern Challenges:** I used nested loops to draw expanding geometric star triangles on the screen.

## What I Learned
This was my first real experience fixing syntax errors. It taught me how to trace loop paths in my head and choose the right loop type for different programming tasks.
