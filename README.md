# C-ADS
This repository contains my foundational work in C programming, developed during my 1st Year of the Common Core Engineering program at Mouloud Mammeri University of Tizi-Ouzou. It covers the transition from basic algorithmic logic to advanced dynamic memory management and data structures.

Key Concepts Covered :
Memory Management: Deep understanding of pointers, dynamic allocation (malloc, free), and passing by reference.  
Data Structures: Custom implementation of Singly Linked Lists, Stacks (LIFO), Queues (FIFO) from scratch.
Algorithmic Logic: Iterative and recursive problem solving (e.g., Fibonacci, Factorials, Prime numbers).

Why this matters for my future ?
Building these structures from scratch in C has provided me with a deep understanding of how memory and data operate at a low level. This rigorous foundation is a stepping stone for my future specialization in computer networks and cybersecurity, where understanding memory allocation and data packet structuring is crucial.
